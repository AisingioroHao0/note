# k近邻法

给定一个训练数据集，对新的输入实例，在训练数据集中找到与该实例最邻近得k个实例，这k个实例多数属于某个类，就把该输入实例分为这个类。

## 模型

对应于特征空间得划分。

由三个基本要素组成：

- 距离度量
- k值
- 分类决策规则

多数表决

分类得损失函数为01损失函数,则经验风险最小化得策略即多数表决。
