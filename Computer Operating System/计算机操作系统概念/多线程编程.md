# 线程库
## Posix Pthreads
```cpp
#include<pthread.h>
void *f(void * param)//线程函数
{
  pthread_exit(0);//线程退出
}
pthread_t tid;//线程标识符
pthread_attr_t attr;//线程的特性
void *p;//参数指针
pthread_create(&tir,&attr,f,p);
pthread_join(tid,NULL);//等待子线程结束
```
编译时需加-lpthread
```bash
g++ main.cpp -o main -lpthread
```
